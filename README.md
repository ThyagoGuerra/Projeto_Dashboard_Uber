# 🚀 Uber Data Analysis Dashboard

Este projeto tem como objetivo analisar dados de corridas de um cenário inspirado na Uber, utilizando SQL para tratamento dos dados e Power BI para visualização.

## 📊 Visão Geral

O dashboard permite acompanhar:
- Faturamento total
- Quantidade de corridas
- Satisfação média dos usuários
- Métodos de pagamento
- Faturamento por cidade
- Evolução mensal

## 🛠️ Tecnologias utilizadas

### SQL
- CASE WHEN
- CTE
- SUBQUERY
- AVG, SUM, ROUND
- GROUP BY, ORDER BY
- CREATE VIEW
- WHERE, LIMIT

### Power BI (DAX)
- AVERAGE, SUM, DIVIDE
- CALCULATE
- COUNT, COUNTROWS
- VAR, SWITCH
- FORMAT
- DATEADD, MONTH
- RETURN

Além de recursos como:
- Tooltips
- Indicadores visuais
- Ícones dinâmicos
